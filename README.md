## Installation

Use the package manager [npm](https://npm.org) to install code.

```bash
npm install
```

You must create a config.js file in root with:

```javascript
const dataOptions = {
  db: {
    host: "localhost",
    database: "miningmars",
    user: "root",
    password: "root",
    port: "3306",
  },
};
module.exports = dataOptions;
```
